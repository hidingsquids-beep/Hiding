# Hiding
Some daily trivial code repositories update some self-built Skills based on projects

## Skills

### operation-manual-workflow

从真实后台链接、Axure 原型、本地原型包、截图或业务说明生成培训可交付的功能操作手册。

适用场景：

- 操作手册、用户手册、培训手册
- 真实后台链接制册、管理后台页面采集
- Axure 原型生成页面操作说明
- 保存、提交、删除、审批、发布等高风险操作前确认

安装示例：

```bash
npx skills add hidingsquids-beep/Hiding --skill operation-manual-workflow
```
